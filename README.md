### Method 1: The Easy Way (Secure)
1.  **Download** this folder to your computer.

### Method 2: The Pro Way (No Popup)

1.  Open the "index.html" file in a text editor (like Notepad or VS Code).
2.  Scroll down to the script section (around Line 173).
3.  Find this line:
    `const HARDCODED_KEY = "";`
4.  Paste your key inside the quotes:
    `const HARDCODED_KEY = "gsk_your_key_here";`
5.  **Save the file.** Now the app will open instantly without asking for a key!
